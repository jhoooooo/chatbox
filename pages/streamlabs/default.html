Get started

This is a custom overlay made for Streamlabs chatbox widget for Twitch and Youtube. It features custom animation, extra functionalities and more.

Features Back to top

  • BTTV, FFZ, 7TV emotes by Streamlabs
  • Option to enable smoothscroll animation
  • Supports Twitch's replies, /clear & /announce
  • Supports Youtube Live chat

Download v2.0 Back to top

Download all the necessary file via Gumroad. All the files can be opened with Notepad. Simply unzip the file, drag and drop the individual file into the notepad app.

Installation Back to top

You can choose between one-click installation or manual installation method.

One-click installation does not support Streamlabs 'Hide-chatter' option. You must disable it.

One-click Installation Back to top

  1. Extract the zip file you downloaded from Gumroad. In the 01 - One-click Installation folder, open up the link shortcut file by double clicking it.
  2. Upon clicking on it, a new tab will open on your web browser directing you to Streamlabs' website.
  3. A popup will appear asking you where to install the chatbox as show in the image below. You can either use default/existing 'collection' or create new theme 'collection'.
  4. You will then be redirected to Streamlabs Chat Box widget page where all the settings and codes automaticly gets installed to.
  5. You can now customize the chat box.

Manual Installation Back to top

  1. Open your Streamlabs web dashboard and proceed to chat widget section.
  2. Look for Enable Custom HTML/CSS and choose the enable option.
  3. After that, few new fields will be available as shown below.
  4. Extract the zip file you downloaded from Gumroad. In the 02 - Manual Installation folder contains the codes needed which is (HTML, CSS, JS & Custom-Field). These code files can be open using notepad or any other text editor. On Streamlabs widget page, click on each tab and replace the content with their respective code. HTML on the HTML tab, CSS on CSS tab and so on.
  5. For Custom Fields, if it is your first time editing this field, you have to click on Add Custom Fields > Edit Custom Fields > paste the code in and then click Update.
  6. The Custom Fields tab should now look like the example shown below.
  7. You can now customize the chat box.

Customization Back to top

  1. On Custom Fields section, you can customize the chat box and make changes as you please.
  2. But, before that, make sure your Streamlabs settings are set as recommended below.

Streamlabs Settings Back to top

  1. Themes Ignore
  2. Badges Choose as you please
  3. Extra Emotes Recommended to enable
  4. Background Color Ignore
  5. Text Color Ignore
  6. Font Size Ignore
  7. Hide Message After Always show messages
  8. Chat Delay Choose as you please
  9. Hide Chatter Recommended to disable
    10. Hide Chatter Option

    If you choose to enable this option, please make sure to use JS-Hide-Chatter file instead of normal JS file in the JS tab. You must also enable both Hide Common Chat Bots & Hide commands starting with `!` option.

    If you use custom made or named bot, you have to find code snippet below in the JS file and insert your bot name(s) into it.

    const botNames = ["nightbot","moobot","xanbot","deepbot","vivbot","phantombot","streamelements"];

    Lastly, you have to disable reply option on 'Custom Field' section as Streamlabs will delete replies to any bots.

  10. Muted Chatters Recommended to disable
  11. Custom Bad Words Recommended to disable
  12. Enable Custom HTML/CSS Enabled

Theme Settings Back to top

Font
You can choose font in the dropdown option or by entering any font name from Google Fonts
Font Size

Font sizes may affect the how other element behave.

Text Line Height
Sets the height of a line box, the distance between lines of text. Bigger font require higher text line height.
Chatbox Background Color
Chatbox background color. You can use 8 digits Hex color for transparency.
Chatbox Padding
Inner spacing between chatbox and it's content. higher resolution looks better with bigger padding.
Chatbox Border Style, Color, Width, Radius
Increase the border width if you want to have a border and tweaks it's style to your own liking.
Message Background Color
Individual messages background color. You can use 8 digits Hex color for transparency.
Username Color
If you prefer different color for people's username.
Message Color, Weight, Border Style, Border Color, Border Width, Border Radius
If you choose to have different background color for individual messages, tinker with these for further customization.
Spacing Between Messages
Spacing between individual messages, and these is not spacing between text lines (line height).
/announce options
To enable Twitch's /announce feature. It has 5 different color option (Primary, Blue, Green, Orange and Purple). If you prefer, you can also hide announcer's username. These messages does not support emotes.
Show replies?
To enable Twitch's replies feature. Does not support 'Hide Chatter' option. If people replied to these hidden account (such as bots), it may ruin the chatbox behaviour. You can limit the length of these replies so that it does not takes to much space.
Smoothscroll Animation
To enable smoothscroll animation each time new messages arrived. Suitable for slow chats.
Limit Message Shown
If you prefer, you can limit how many messages displayed at a time.

Color Picker Back to top

8 digits hex color picker. Use the slider to set both color and it's transparency level.
