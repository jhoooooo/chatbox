Streamlabs Chat Box Overlay Installation Guide

Last Update 03 April 2022
This installation guide is for Streamlabs web dashboard, not Streamlabs Desktop (SLOBS) application. Make sure you have downloaded the theme files from Gumroad.
Each themes might have different options. Not all steps are required or applicable to each themes.

Streamlabs Settings

Visit your Streamlabs web dashboard and proceed to chat widget section. Setup your options as shown below.
  1. Themes Ignore
  2. Badges Choose as you please
  3. Extra Emotes Recommended to enable
  4. Background Color Ignore
  5. Text Color Ignore
  6. Font Size Ignore
  7. Hide Message After Always show messages
  8. Chat Delay Choose as you please
  9. Hide Chatter Recommended to disable
    10. Hide Chatter Option

    If you enable this option, later please make sure to use Themename-JS-Hide-Chatter file instead of Themename-JS. You must also enable both Hide Common Chat Bots & Hide commands starting with `!` option.

    If you use custom made/named bot, you have to find code snippet below in the JS file and add in your bot name(s).

    const botNames = ["nightbot","moobot","xanbot","deepbot","vivbot","phantombot","streamelements"];
  10. Muted Chatters Recommended to disable
  11. Custom Bad Words Recommended to disable
    12. These options should not effect anything. If you have any problems, please contact me.
  12. Enable Custom HTML/CSS Enabled

Theme Settings

  1. After enabling Custom HTML/CSS, new field should appear like the image shown below.
  2. You can now open/extract the zip file. It contains the codes needed (HTML, CSS, JS & Custom-Field). These code files can be open using notepad.
  3. On the widget page, click on each tab and replace the content with their respective code. HTML on the HTML tab, CSS on CSS tab and so on.
  4. For Custom Fields, if it is your first time editing it, you have to click on Add Custom Fields > Edit Custom Fields > paste the code in and then click Update.
  5. The Custom Fields tab should now look like the example shown below.
  6. You can modify each options in Custom Fields tab to customize your chatbox visual. This project is here to help you create your own unique theme. Take your time customizing it. Ask for help if needed.
    7. Tips: You can add transparency to color by using 8 digits hex color.
    Tips: Don't just randomly choose a color. It is recommended to have a set of color palette for your whole stream overlay. And then match those while making this chat box so that they compliment each other.
    • Happy Hue Palettes
    • Coolors Palettes Generators
  7. Click Save Settings to save your theme.
  8. Finally, you can copy the widget url located on top of the page and use it as browser source in your OBS.
  9. For testing purposes, you can add ?simulate=1 after your widget url to simulate users.
  10. Example: https://streamlabs.com/widgets/chat-box/v1/XXXCODEXXXXX?simulate=1
